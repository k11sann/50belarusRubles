////////////// РУС //////////////
Вы можете создавать свои прессеты на граффити

1. Добавьте свою текстуру в папку 'brushTextures'

2. Откройте PresetSettings.json [ можно открыть как .txt ]

3. В PresetSettings.json вы можете менять цвета ["SprColors", hex color ];
А также вы можете добавлять имена для текстур балончика [ имя текстуры как имя текстуры в 'brushTextures' ]

Размер текстуры 170х170 мяу, можете больше ;)

* PresetUnuse = Хранить ваши пресеты [ ТОЛЬКО ХРАНИТЬ, ЭТА ПАПКА НЕ ИСПОЛЬЗУЕТСЯ НИКАК ! ]
